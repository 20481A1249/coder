CSS

- The fullform of CSS is: Cascading Style Sheets

- There are 3 ways to insert css:
1.Inline CSS
2.Internal CSS
3.External CSS

1.Inline CSS: By using style attribute inside the HTML elements.

Example:
     < h1 style="color:red;">A Blue Heading< /h2>

2.Internal CSS: By using a < style > element in the < head > section.

Example:
    < style>
        body{
            background-color: green;
            color:blue;
            color:red;
        }
    < /style>

3.External CSS: By using a < link > element to link to an external CSS file.

Exampe:
    < link rel="stylesheet" href="styles.css">

Priority:
1.Inline CSS
2.Internal and External CSS depends on the updated tag in the HTML. 

Note:
    p{
        color:red !important; // !important adds priority weightage.
    }

CSS Selectors:

Selectors are used to find or pointout the HTML elements you want to style.

Selectors are divided into 5 categories:
1.Simple selectors
2.Combinator selectors
3.Pseudo-class selectors
4.Pseudo-elements selectors
5.Attribute selectors

1.Simple Selectors:
1) class selector (.)
2) Id selector (#)
3) Element selector
4) Grouping selector
5) Universal selectors (*)

1) class selector: The class selector selects HTML elements with a specific class attribute.

Example:
    .center{
        text-align: center;
        color: red;
    }

2) Id selector: The id of an element is unique within a page, so the id selector is used to select one unique element.

Example:
    #id{
        text-align: center;
        color: red;
    }

3) Element selector: The element selector selects HTML elements based on the element name.

Example:
    p{
        text-align: center;
        color: red;
    }

4) Grouping selector: The grouping selector selects all the HTML elements with the same style sefinitions.

Example:
    h1, h2, p{
        text-align: center;
        color: red;
    }

2) Universal selector: The Universal selector selects all HTML elements on the page.

Example:
    *{
        text-align: center;
        color: red;
    }
